# 보일러플레이트 생성 지침

**목적**: 특정 연도의 분기 단위 문서가 없을 때, 링크·요약 채우기 전의 **빈 분기 문서 골격**을 생성함.

## 적용 시점

- 사용자가 "○년도 분기 단위 보일러플레이트 생성해줘", "○년 분기별 보일러플레이트 추가" 등으로 요청한 경우
- **있는 것은 그대로 둠**: 이미 존재하는 분기 파일(예: `2023-10-01_to_2023-12-31.md`)은 덮어쓰지 않고, 없는 분기만 새로 생성

## 파일 명명·위치

- **경로**: `docs/quality-updates/{연도}/{연도}-MM-DD_to_{연도}-MM-DD.md`
- **분기별 구간** (표준):
  - Q1: `01-01` ~ `03-31`
  - Q2: `04-01` ~ `06-30`
  - Q3: `07-01` ~ `09-30`
  - Q4: `10-01` ~ `12-31`

## 필수 구조

1. **YAML frontmatter** (기존 분기 문서와 동일 형식)
   - `title`, `description`, `jurisdiction: KR`, `year`, `frequency: quarterly`, `period_label`(예: 2023-Q1), `period.start`/`period.end`, `category`, `agencies`, `tags`
2. **Executive Summary**: 1문장 플레이스홀더 (예: "본 기간의 주요 동향은 추후 Phase 2 요약 시 기재 예정임.")
3. **기관별 요약**: `!!! success` + `=== "금융감독원"` 등 4개 기관 탭, 각 탭에 `- (추후 기재)`
4. **시사점**: `!!! success` + `=== "기업"` / `=== "감사인"` 탭, 각 `- (추후 기재)`
5. **기관별 본문 섹션** (헤더만 유지, 링크 없음):
   - **금융감독원**: `#### 보도자료`, `#### 세칙제ㆍ개정예고`, `#### 회계감독 동향자료` → 각 "해당사항 없음"
   - **금융위원회**: `#### 보도자료`, `#### 고시/공고/훈령`, `#### 입법예고/규정변경예고` → 각 "해당사항 없음"
   - **한국공인회계사회**: `#### 알림마당 - 공지사항`, `#### 회계감사 - 감사인증기준` → 각 "해당사항 없음"
   - **한국회계기준원**: `#### 소통광장 - 공지사항`, `#### 소통광장 - 보도자료`, `#### 주요일정` → 각 "해당사항 없음"
