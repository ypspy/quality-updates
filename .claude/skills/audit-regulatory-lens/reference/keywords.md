# assertion → 검색 키워드

`focus_areas`·조서 맥락에서 repo fallback `rg` 검색 시 사용. 동의어는 OR(`|`)로 묶어 검색한다.

| focus_area | 검색 키워드 (ko) | 기관 힌트 |
|------------|------------------|-----------|
| ICFR / 내부회계 | 내부회계, 운영실태, 자금 부정, ICFR, 통제활동 | FSS, KICPA |
| 수익인식 | 수익인식, 매출, 진행률, 계약자산 | FSS |
| 전환사채 | 전환사채, CB, 전환권, 콜옵션, 풋옵션 | FSS, FSC |
| 투자자약정 | 투자자약정, 투자자 약정, 금융부채 분류 | FSS |
| 공급자금융 | 공급자금융, 역팩토링, 공급자 금융 | FSS |
| 관계기업·손상 | 관계기업, 지분법, 손상, 회수가능액 | FSS |
| 특수관계자 | 특수관계자, 특관자, 저가 매각 | FSS, FSC |
| 감사품질·감리 | 감사품질, 감사인감리, 감사인 지정, 투입시간 | FSS, KICPA |
| 독립성·비감사 | 독립성, 비감사용역, 네트워크 회계법인 | FSS, KICPA |
| 사업보고서·공시 | 사업보고서, 결산, 유의사항, 주석 | FSS, KICPA |
| 제재·지적사례 | 조사·감리, 과징금, 지적사례, 회계부정 | FSS, FSC |
| 외부감사 방해 | 외부감사 방해, 감리방해, 자료제출 | FSS, FSC |
| K-IFRS·기준 | K-IFRS, 회계기준, 기준서 | KASB, FSS |
| 보험·금융 | IFRS17, K-ICS, 금융회사, 보험 | FSS |

**사용 예**

```bash
rg -i "전환사채|CB|특수관계자" docs/quality-updates/2025/ docs/quality-updates/2026/
rg -i "감사품질|감사인감리|독립성" docs/quality-updates/2025/
```
