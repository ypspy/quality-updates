# 규제 업데이트

금융위원회, 금융감독원, 한국공인회계사회, 회계기준원 등 주요 기관의  
보도자료·제도 개정·감독정책 변화를 **연도·분기별**로 정리한 문서입니다.

**아래 탐색** 또는 좌측 목차에서 연도 → 분기를 선택하면 해당 기간 업데이트로 이동합니다.

---

## 연도별 문서

- **[2025년](2025/2025-10-01_to_2025-12-31.md)** — 1·2·3·4분기 (01–12월)
- **[2024년](2024/2024-10-01_to_2024-12-31.md)** — 1·2·3·4분기 (01–12월)
- **[2023년](2023/2023-10-01_to_2023-12-31.md)** — 2·3·4분기 (04–12월, 세부 구간별)
- **[2022년](2022/2022-12-15_to_2023-04-03.md)** — 2022-12 ~ 2023-04

각 문서에는 해당 기간의 규제 변화 요약, 기관별 보도자료·의견수렴·감리 결과 링크가 포함됩니다.
