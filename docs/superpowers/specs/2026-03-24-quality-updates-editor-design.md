# Quality Updates Editor — 로컬 큐레이션 도구 설계 스펙

**날짜**: 2026-03-24
**상태**: 승인됨
**범위**: quality-updates 레포 내 로컬 편집 도구

---

## 개요

크롤러가 생성한 `.md` 파일을 Claude Code에 넘기기 전, 링크 선별(요약 필요/스킵)과 PDF 연결을 체계적으로 수행하기 위한 로컬 웹 편집 도구.

**목적**: 링크 큐레이션·PDF 지정 작업을 UI로 처리하고, 결과를 `.md` 주석으로 저장 → Claude Code가 SKILL.md Phase 0·1 수행 시 참조

**범위 외**: Claude API 직접 호출, PDF 자동 다운로드, 배포·팀 공유

## 아키텍처

### 파일 구조

```
scripts/
  editor.py              # Flask 앱 진입점
  editor_config.json     # 설정 영속화 (다운로드 폴더 경로 등)
  editor/
    templates/
      index.html         # 메인 UI (Jinja2 템플릿)
    static/
      editor.js          # 인터랙션 (상태 토글, iframe 로드, 저장)
      editor.css         # 최소 스타일
```

### 실행

```bash
python scripts/editor.py
# → http://localhost:5000 자동 오픈
```

### 의존성 추가 (requirements.txt)

```
flask
```

```

    ↓

    ↓
브라우저: 좌측 패널에 링크 목록 표시
    ↓

    ↓
저장 버튼
    ↓
Flask: .md.bak 백업 → 주석 삽입 후 원본 덮어씀
    ↓
Claude Code: 주석 참조하여 Phase 0·1 수행
```

## UI 설계

### 레이아웃

좌우 분할 패널 (splitter로 너비 조절 가능)

```
┌─────────────────────────────────────────────────────────┐
│ [파일 선택 ▼]  다운로드 폴더: [경로표시] [변경]          │
│ 전체 42 | 요약 필요 12 | 스킵 8 | 미결정 18 | 완료 4   [저장] │
├──────────────────────────┬──────────────────────────────┤
│ 날짜  제목  기관  상태  PDF │  iframe / fallback 버튼     │
│ ...                      │                              │
└──────────────────────────┴──────────────────────────────┘
```

### 헤더

| 요소 | 설명 |
|------|------|
| 파일 선택 드롭다운 | `docs/quality-updates/` 아래 `.md` 파일 목록 |
| 다운로드 폴더 경로 | `editor_config.json`에서 로드. 없으면 `downloads/` 기본값 |
| 변경 버튼 | 폴더 경로 직접 입력 또는 브라우저 경유 선택 |
| 진행 현황 카운터 | 전체 / 요약 필요 / 스킵 / 미결정 / 완료 실시간 표시 |
| 저장 버튼 | 변경사항을 `.md`에 반영 |

### 좌측 패널: 링크 목록 테이블

기관별 섹션 헤더(금융감독원 / 금융위원회 / 한국공인회계사회 / 한국회계기준원)로 구분.

| 컬럼 | 내용 |
|------|------|
| 날짜 | YY-MM-DD |
| 제목 | 클릭 시 우측 패널에 해당 URL 로드 |
| 기관 | 섹션 기반 자동 태깅 |
| 상태 | 클릭 순환: **미결정 → 요약 필요 → 스킵 → 미결정** |
| PDF | 다운로드 폴더 파일 드롭다운 (없으면 비활성화) |

**상태별 행 스타일**:
- 미결정: 기본
- 요약 필요: 파란색 강조
- 스킵: 흐린 회색
- 완료(이미 `!!! note` 존재): 회색 읽기 전용, 수정 불가

### 우측 패널: 원문 뷰어

1. 제목 클릭 시 `<iframe src="[URL]">` 로드 시도
2. `load` 이벤트 후 iframe 내부 접근 실패(X-Frame-Options 차단) 감지
3. 차단 감지 시: iframe 숨기고 "이 사이트는 분할 화면을 지원하지 않습니다" 메시지 + **새 탭에서 열기** 버튼 표시

## `.md` 주석 포맷

### 스킵 항목

```markdown
- (25-01-15) [제목](https://example.com)
<!-- skip -->
```

### PDF 지정 항목

```markdown
- (25-01-20) [제목](https://example.com)
<!-- pdf: downloads/250120_보도자료.pdf -->
```

### 주석 없음 = 미결정

```markdown
- (25-01-25) [제목](https://example.com)
```

### 완료 항목 (에이전트 처리 후)

```markdown
- (25-01-25) [제목](https://example.com)

    !!! note "주요 내용"

        - ...
```

## Flask 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/` | 메인 UI 렌더링 |
| GET | `/api/files` | `.md` 파일 목록 반환 |
| GET | `/api/links?file=<path>` | 파일 파싱 → 링크 목록 JSON |
| GET | `/api/downloads` | 다운로드 폴더 파일 목록 |
| POST | `/api/save` | 큐레이션 결과를 `.md`에 저장 |
| POST | `/api/config` | 다운로드 폴더 경로 저장 |

## 파싱 전략

### Appendix 경계 처리

파일을 라인 단위로 순회하되, `^## Appendix` 패턴의 헤더를 만나는 순간 이후 라인은 **파싱 및 저장 대상에서 완전히 제외**한다. Appendix A에는 크롤러 원시 링크(비필터링)가 수백 개 포함되어 있어, 큐레이션 대상이 아님.

### 링크 추출 정규식

```python
LINK_PATTERN = re.compile(
    r'^(\s*- \()(\d{2}-\d{2}-\d{2})\) \[(.+?)\]\((https?://[^\)]+)\)(.*)'
)
```

Appendix 헤더 이전 라인에만 적용. 저장 시에도 동일한 경계 내에서만 주석 삽입/삭제.

### 상태 복원 (파일 로드 시)

- 링크 다음 줄 `<!-- skip -->` → 스킵
- 링크 다음 줄 `<!-- pdf: ... -->` → 요약 필요 + PDF 경로 복원
- 링크 다음 줄(비어있지 않고) `!!! note` 또는 `??? note` → 완료

### 저장 전략

원본 텍스트를 전체 재조립하지 않고, **라인 단위로 주석만 삽입/삭제**:
- 기존 `<!-- skip -->` / `<!-- pdf: ... -->` 주석 제거 후 재삽입
- `!!! note` 블록은 절대 수정하지 않음

## 설정 영속화

`scripts/editor_config.json`:

```json
{
  "downloads_folder": "downloads/",
  "last_file": "docs/quality-updates/2025/2025-01-01_to_2025-03-31.md"
}
```

- 앱 시작 시 로드, 없으면 기본값으로 생성
- 다운로드 폴더 변경·파일 선택 시 자동 업데이트

## 안전장치

- 저장 전 `.md.bak` 자동 생성 (동일 디렉터리)
- 파싱 실패 링크는 건너뛰고 원문 유지
- `downloads/` 폴더 없으면 PDF 드롭다운 비활성화 (오류 없이)
